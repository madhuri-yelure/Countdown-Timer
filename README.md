# Countdown Timer

A responsive Countdown Timer web application built using HTML, CSS, and JavaScript that displays real-time countdowns in days, hours, minutes, and seconds until a specified event through a clean and user-friendly interface.

## Features

- Real-time countdown timer
- Displays days, hours, minutes, and seconds
- Event-based countdown
- Responsive and modern user interface
- Easy to use

## Technologies Used

- HTML
- CSS
- JavaScript
